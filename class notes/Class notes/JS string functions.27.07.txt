

STATIC vs NON-STATIC methods:
  -static methods are invoked with the Object/Class/Blueprint templates
  -non static methodes are invoked with the instance of the Object/Class/Blueprints methods

Math.max(5, 3) // static method
Number.isInteger(5); // static method

let number = 25.556443;
number.toFixed(2); //25.55


toFixed() is NON-STATIC function


'ab'[0];   // 'a'
'hi'.at(-3);  // undefined
'good morning'.at(-3); // 'i'
